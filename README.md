# Cybertron
App that scores a war of Autobots and Decepticons.
[Api docs](https://transformers-api.firebaseapp.com/api-docs/)

[![xCode](https://img.shields.io/badge/xCode-10.0-red.svg)](https://img.shields.io/badge/xCode-9.3-red.svg) [![Swift4](https://img.shields.io/badge/Swift-4.2-FFAC45.svg)](https://img.shields.io/badge/Swift-4.2-FFAC45.svg)


### Usage
Clone the repo

Make sure you have Cocoapods installed

Run ```pod install``` from the repository folder (in your shell/Terminal)

Open the .xcworkspace file in XCode

Build the project

### Libs used

* [Moya](https://github.com/Moya/Moya)
* [AlamofireImage](https://github.com/Alamofire/AlamofireImage)
